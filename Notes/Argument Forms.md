#philosophy

---

### Modus Ponens

If A then B
A
Therefore, B

---

### Modus Tollens

If A then B
!B
Therefore, !A

---

### Hypothetical Syllogism

If A then B
If B then C
Therefore if A then C

---

### Disjunctive Syllogism

A xor B
!A
therefore B

---

### Dilemma

A or B
If A then C
If B then D
Therefor C or D
